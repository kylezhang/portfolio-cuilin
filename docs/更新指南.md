## 快速开始 - 3 步上传作品

### 第一步：放图片
把作品图片放到 `static/images/` 下的分类目录中，建议命名格式：
```
static/images/green/001-九号单车.jpg
static/images/edu/001-知行学堂.jpg
```

### 第二步：填数据
编辑 `data/data.json`，添加作品信息：
```json
{
  "category": "绿色出行",
  "categoryDesc": "品牌视觉系统",
  "span": "2x2",
  "works": [
    { "title": "九号单车品牌系统", "image": "static/images/green/001-九号单车.jpg" }
  ]
}
```

- `category`：分类名称
- `categoryDesc`：分类描述
- `span`：`2x2` 是大图作品，否则为普通尺寸
- `works`：该分类下的作品列表（多个作品会自动轮播）

### 第三步：发布
在终端运行：
```bash
bash static/scripts/sync.sh "新增了某某作品"
```

## 常见问题

**Q: 如何修改首页大图位置？**  
A: 在 `data/data.json` 中，将 `span: "2x2"` 的作品放在数组第一个位置即可。

**Q: 图片不显示怎么办？**  
A: 检查图片是否在 `static/images/` 目录，`data/data.json` 中的路径是否正确（如 `static/images/green/001-xxx.jpg`）。

**Q: 能否不修改代码？**  
A: 是的！只需编辑 `data/data.json` 和添加图片即可，无需修改 HTML 代码。
