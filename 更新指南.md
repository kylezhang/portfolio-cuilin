# 作品集网站 - 内容更新指南

> 给设计师崔琳的维护文档

---

## 网站结构说明

整个网站就是一个文件：`portfolio.html`。所有内容都在这个文件里。

## 更新作品内容

打开 `portfolio.html`，找到 `<!-- WORK -->` 注释区域，你会看到类似这样的结构：

```html
<div class="work-item reveal">
    <div class="work-placeholder work-cat-1">
        <span class="work-placeholder-label">[ 封面图 16:9 ]</span>
    </div>
    <div class="work-overlay">
        <h3>九号单车</h3>
        <p>绿色出行 · 品牌视觉系统</p>
    </div>
</div>
```

**修改项目名称**：改 `<h3>` 里的文字
**修改分类描述**：改 `<p>` 里的文字
**替换图片**：把 `div.work-placeholder` 替换为真实图片：
```html
<img class="work-thumb" src="images/项目名.jpg" alt="项目名称">
```

## 更新个人介绍

找到 `<!-- ABOUT -->` 区域，修改 `<p>` 标签里的文字即可。

## 更新联系方式

找到 `<!-- CONTACT -->` 区域，修改邮箱链接和社交链接。

## 发布更新

在终端中运行：
```bash
bash sync.sh "更新了某某作品内容"
```

Vercel 会在推送后自动部署，约 1-2 分钟后访问 `https://qlink.ai-trainer.fun` 即可看到更新。
