```

├── index.html          # 网站页面

│   └── ...
└── sync.sh             # 一键发布脚本
```

---

## 快速开始 - 3 步上传作品

### 第一步：放图片
把作品图片放到 `images/` 目录，建议命名格式：
```
001-九号单车.jpg
002-城市马拉松.jpg
003-知行学堂.jpg
```

### 第二步：填数据
编辑 `data.json`，添加作品信息：
```json
{
  "title": "作品名称",
  "category": "类别 · 描述",
  "image": "images/001-九号单车.jpg",
  "span": "1x1"
}
```

- `title`：作品名称
- `category`：分类和描述
- `image`：图片路径（在 `images/` 目录下的文件）
- `span`：`1x1` 是普通作品，`2x2` 是大图作品（首页第一个位置）

### 第三步：发布
在终端运行：
```bash
bash sync.sh "新增了某某作品"
```

## 常见问题

**Q: 如何修改首页大图位置？**  
A: 在 `data.json` 中，第一个作品会自动显示在大图位置（2x2）。

**Q: 图片不显示怎么办？**  
A: 检查图片是否在 `images/` 目录，`data.json` 中的路径是否正确。

**Q: 能否不修改代码？**  
A: 目前需要简单编辑 `data.json`（JSON 格式很直观，像表格一样）。
